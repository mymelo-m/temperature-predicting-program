# Temperature Predicting Program

Based on historical weather data from Shanghai for 2013–2022, this project consists of a temperature prediction model. Using SVR and Random Forest regression algorithms, it analyzes the impact of various meteorological factors on temperature changes and compares the predictive performance of different models.
The five different versions of SVR take different variables that affect the weather into account. The details are given in the corresponding results.
